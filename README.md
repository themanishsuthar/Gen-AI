# 🧠 Advanced Personality Profiler

An AI-powered **Advanced Personality Profiler** that analyzes your personality traits based on your introduction, work style, decision-making approach, social battery, and stress response.

This application uses **LLM-powered insights** to generate a thoughtful personality profile using **Google Gemini 2.5 Flash** with **LangChain** and an intuitive **Streamlit UI**.

---

# 🚀 Features

* Paste your personal introduction
* Select your **Work Style**
* Choose your **Decision Making** approach
* Define your **Social Battery**
* Describe your **Stress Response**
* AI-generated personality analysis
* Clean and interactive UI using Streamlit

---

# 🧩 Parameters Used

### 1. Work Style

Choose one:

* Deep Focus / Solo
* Collaborative / Team
* Fast-paced / Dynamic

---

### 2. Decision Making

Choose one:

* Purely Emotional
* Balanced
* Data-Driven

---

### 3. Social Battery

Choose one:

* Introverted (Recharge Alone)
* Extroverted (Recharge with Others)

---

### 4. Stress Response

User Input Field:

**Question:**
How do you react to tight deadlines?

**Placeholder Example:**
Eg: "I get organized" / "I get anxious"

# ⚙️ Installation

### 1. Clone Repository

```
git clone https://github.com/themanishsuthar/advanced-personality-profiler.git
```

### 2. Navigate to Project

```
cd advanced-personality-profiler
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Add Environment Variables

Create `.env` file

```
GOOGLE_API_KEY=your_api_key_here
```

---

# ▶️ Run Application

```
streamlit run app.py
```

# 🎯 How It Works

1. User enters introduction
2. Selects personality parameters
3. Inputs stress response
4. Data sent to LLM via LangChain
5. Google Gemini 2.5 Flash analyzes inputs
6. AI generates personality profile
